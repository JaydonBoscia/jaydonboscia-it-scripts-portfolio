# Log Analyzer Script

## Overview
This script analyzes log files to identify error and warning patterns
commonly encountered in application and system troubleshooting.

## Features
- Scans log files for error and warning messages
- Counts and ranks the most frequent issues
- Generates a summarized report for faster diagnosis

## Use Cases
- Application support troubleshooting
- Server log review
- Incident analysis

## Skills Demonstrated
- Python scripting
- Application support
- Systems analysis
- Log parsing and diagnostics
