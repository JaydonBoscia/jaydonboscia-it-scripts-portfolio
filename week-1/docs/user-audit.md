# User Audit Script

## Overview
This script performs a basic audit of a workstation or server by collecting
system and user information to assist IT support and systems administrators.

## Functions
- Collects operating system and hardware information
- Identifies the currently logged-in user
- Lists local user accounts

## Practical Use
- Help Desk troubleshooting
- New system onboarding
- Systems and server audits

## Skills Demonstrated
- Python scripting
- Systems analysis
- User support
- Cross-platform administration
